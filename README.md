# GITit

A simple CLI tool to automate repo creation and initialization.

## Tools
- Python3
- Click
- PyGitHub
- gitpython

## Usage
### 1. Installation
```bash git clone https://github.com/nitishp25/GITit```
```bash cd GITit```
### 2. Get Dependencies
```bash pip install -r requirements.txt```
### 3. Run Script
```bash python3 main.py```
